# UIデザイン改善：ダッシュボードカードとサイドバーのスタイリング

## 何を変えたか

| ファイル | 変更内容 |
|---|---|
| `app/(main)/dashboard/page.tsx` | サマリーカードの色・枠線・文字色を改善 |
| `components/ui/Sidebar.tsx` | サイドバーに `z-index` とドロップシャドウを追加 |

---

## 1. Tailwind CSSの透明度付き背景色

### 書き方

```tsx
// bg-{色}-{濃さ}/{透明度(0〜100)}
<div className="bg-green-700/35">  // 緑 + 35%の不透明度
<div className="bg-stone-200/60">  // グレー系 + 60%の不透明度
```

透明度を使うと、背景画像（壁紙テクスチャなど）が透けて見えるようになります。

**ポイント：** 数値が小さいほど透明（薄い）、大きいほど不透明（濃い）。

---

## 2. カードを「濃い/薄い」で交互に並べる

今回のダッシュボードでは、`dark: true/false` というフラグをデータに持たせて、条件分岐でスタイルを切り替えています。

```tsx
const cards = [
  { label: '累計擬似報酬', bg: 'bg-green-700/35', dark: true },
  { label: '納品完了件数', bg: 'bg-stone-200/60', dark: false },
  { label: '進行中',       bg: 'bg-green-700/35', dark: true },
  { label: '受注総数',     bg: 'bg-stone-200/60', dark: false },
]

// JSXでの条件分岐
<span className={`text-sm ${card.dark ? 'text-green-900' : 'text-stone-500'}`}>
  {card.label}
</span>
<div className={`text-2xl font-bold ${card.dark ? 'text-green-950' : 'text-stone-800'}`}>
  {card.value}
</div>
```

**なぜこうするか：**
- ハードコードで4枚分書くより、データ＋ループのほうが保守しやすい
- フラグ1つで「このカードは濃いスタイル」「このカードは薄いスタイル」を管理できる

## 3. 文字の視認性（コントラスト）

薄い背景色の上に白い文字を乗せると読みにくくなります。

```tsx
// NG: 薄いグリーン背景 + 白文字 → 読みにくい
<div className="bg-green-700/35 text-white">¥45,000</div>

// OK: 薄いグリーン背景 + 濃いグリーン文字 → 読みやすい
<div className="bg-green-700/35 text-green-950">¥45,000</div>
```

**覚え方：** 背景が薄いなら文字は濃く、背景が濃いなら文字は明るく。

Tailwindのグリーン系の濃さ一覧（数字が大きいほど濃い）：
```
text-green-100  // とても薄い（明るい）
text-green-500  // 中間
text-green-700  // やや濃い
text-green-900  // 濃い
text-green-950  // 最も濃い
```

## 4. 枠線（border）の色と透明度

```tsx
// 濃いカードの枠線: グリーン系・薄め
border-green-600/20

// 薄いカードの枠線: グレー系・薄め
border-stone-300/50
```

枠線も透明度を使うことで、背景と自然に馴染みます。`/20` のように薄くすると「なんとなく区切りがある」程度のさりげない枠線になります。

## 5. z-index（重なり順）とドロップシャドウ

### 問題：影が見えない

サイドバーに影をつけても、隣のメインコンテンツの上に描画されていたため、影が隠れて見えませんでした。

### 解決：z-indexで重なり順を制御

```tsx
// Sidebar.tsx
<aside className="... relative z-20 shadow-[4px_0_16px_rgba(0,0,0,0.18)]">
```

- `z-20`：サイドバーをコンテンツより手前に表示
- `shadow-[...]`：Tailwindの任意値（角括弧内にCSSをそのまま書ける）

### ドロップシャドウの読み方

```
shadow-[4px_0_16px_rgba(0,0,0,0.18)]
         ↑   ↑   ↑    ↑
     X方向 Y方向 ぼかし 色(黒・透明度18%)
```

アンダースコア `_` はTailwindの任意値でスペースの代わりに使います。

---

## 6. 学習まとめ

| 技術 | 使いどころ |
|---|---|
| `bg-{色}/{透明度}` | 背景テクスチャを透かしたいとき |
| 条件分岐スタイル | データに応じてカードの色を切り替えるとき |
| コントラスト設計 | 背景と文字の濃淡を対にして視認性を確保 |
| `border-{色}/{透明度}` | 馴染みのいい枠線を作るとき |
| `z-index` | 要素の重なり順を制御して影を見せたいとき |
| `shadow-[...]` 任意値 | Tailwindに存在しない細かい影の値を指定したいとき |
